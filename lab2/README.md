# Telegram To-Do Bot (CSV версия)

## Запуск

pip install -r requirements.txt  
python bot.py  

## Команды

/add — добавить задачу  
/list — список задач  
/search текст — поиск  
/today — задачи за сегодня  

## Описание

Бот работает с CSV файлом:
- сохраняет задачи
- ищет по ключевым словам
- фильтрует по дате
