# Telegram To-Do Bot

## Установка

1. Установите Python 3.10+
2. Установите зависимости:

pip install -r requirements.txt

3. Создайте файл .env и вставьте токен:

BOT_TOKEN=ваш_токен

## Запуск

python bot.py

## Команды

/start — запуск бота  
/help — помощь  
/add — добавить задачу  
/list — список задач  
/done N — удалить задачу  

## Описание

Бот позволяет:
- добавлять задачи
- задавать приоритет
- получать напоминания
- управлять списком задач
